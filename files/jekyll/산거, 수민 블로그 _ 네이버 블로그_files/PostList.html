    그래서 많이 안씀에도 불구하고 계속 멜론 스트리밍 서비스를 유지했어요. 5년 이상!!

    중간에 다른 서비스로 변경할 수 있는 순간들도 많았어요! 네이버 뮤직 3개월 이용권이라던지 (아직도 안쓰는 중)

    그럼에도 불구하고 (무료쿠폰 등) 멜론 서비스를 유지한 이유는

    1. 카카오톡 편리한 연동 (+ 3개월에 한번씩 이모티콘도 줌 (기간 제약) )

    2. 멜론 DJ / 선호하는 앨범 / 곡 변경 어려움

    3. 익숙한 UI

    등 때문에, 아무리 뭐가 됐던간에 옮기지 않았어요. 저런게 은근 귀찮더라구요.

    그러다 결정적으로 해지하게 된 이유는 다름아닌 "가격"입니다. 멜론이 제일 비쌉니다.

    5년 중 4년동안은 부모님이 통신비를 내주기 때문에 별 생각없이 썼었는데요! (야호) 최근에 제 통장으로 변경하고, 요즘 구독 모델이 트렌드다 보니 돈들이 굴러 굴러 조금 커지더라구요. (왓챠 / 넷플릭스 / 애플 / aws 등 )

    저는 노래도 많~이듣는편이 아니라 8500원이 넘는 가격을 부담하는건 좀 돈이 아깝다고 생각이 들더라구요.

    1년이면 10만원 넘는데, 일본 티켓값 정도 되겠네요.

    왓챠 / 넷플릭스를 다 안썼으면 그냥 쓰루 할텐데 둘 중 하나를 포기 못하겠더라구요 ㅋㅋㅋ (구독 이용요금 10만원 돌파..)

    다른건 일상에서 필수적으로 사용해야하는 (notion, apple clouds, aws 등) 제품이었기 때문에, 줄일 필요가 있었어요.
